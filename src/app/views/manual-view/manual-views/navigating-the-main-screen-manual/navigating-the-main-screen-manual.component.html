There are 5 panels/sections in the main view. The top section, no. 1 is the toolbar. Here you will find the file and edit menus. You can also access the settings view (WIP), the exports view and this manual from here. You can also control playback of the preview window, record all tracks live and also rename the project.

The left panel - no. 2 - contains buttons that allow you to toggle the visibility of all other panels.

The panel to the right of this, no. 3, contains the imports panel and add filters panel. Here you can import tracks and select filters to add to the currently selected track.

The right panel, no. 4, contains track panel. Here you can add, remove and edit tracks and clips.

The bottom section, no. 5, contains the timeline. This is where you can add, remove and edit clips. The bottom panel is the information panel and the track properties panel. The properties panel allows you to change the layer filter and the filters being applied to the track. The information panel (WIP) will eventually give you information about the current panel you are hovering over.
